\section{VietVibe - MultiAgent Smart Tourism System}
\subsection{Abstract}
\begin{frame}{Abstract}
    \begin{block}{VietVibe}
    \textbf{VietVibe} is a \textbf{Smart Tourism System} built on \textit{Multi-Agent System} (MAS) architecture, focused on helping tourists discover the \textit{cultural} depth of Vietnam - from street food hidden in narrow alleys, weekly local events, live music shows, to the regional cultural stories behind each destination.
    \end{block}
\end{frame}

\begin{frame}{Abstract}
    \begin{block}{Main features}
    Unlike another conventional tourism applications that \textbf{merely} list popular attractions, VietVibe employs a \texttt{multi-agent} system to:
    \begin{itemize}
        \item \textbf{Automatically collect} data from multiple sources (e.g. Google Maps, Facebook Events, ...)
    \end{itemize}
    \end{block}
\end{frame}
